🌿 TeeNaturals Backend API

A scalable backend API powering the TeeNaturals e-commerce platform. The application manages user authentication, product management, order processing, and administrative operations through a modular RESTful architecture.

The backend is designed to support modern online shopping experiences while remaining easy to maintain and extend.

---

🚀 Features

- User Authentication
- Product Management
- Order Processing
- Admin Operations
- Email Notifications
- RESTful API
- MongoDB Integration

---

🛠 Tech Stack

Backend

- Node.js
- Express.js
- MongoDB
- Mongoose

Services

- JWT Authentication
- Nodemailer
- Express Middleware

🏗 Architecture

Routes → Controllers → Services → Database

The backend follows a modular architecture that separates concerns and improves maintainability.

---

🌟 Why I Built This

This project serves as the backend infrastructure for TeeNaturals, enabling secure user authentication, product management, and order processing for an online natural skincare store.

The objective was to build a reliable and scalable API capable of supporting real-world e-commerce operations.
